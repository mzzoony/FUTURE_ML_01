# Datasets

This directory contains small sample datasets used in projects.

## Important Notes

- **Only small datasets** should be committed here (< 10 MB)
- Large datasets should be:
  - Downloaded programmatically in notebooks
  - Referenced with links to external sources
  - Stored in cloud storage (Google Drive, AWS S3, etc.)

## Dataset Sources

Common sources for ML datasets:
- [Kaggle](https://www.kaggle.com/datasets)
- [UCI ML Repository](https://archive.ics.uci.edu/ml/index.php)
- [Google Dataset Search](https://datasetsearch.research.google.com/)
- [Scikit-learn Built-in Datasets](https://scikit-learn.org/stable/datasets.html)

## Usage

Datasets used in projects will be documented in their respective project README files.
